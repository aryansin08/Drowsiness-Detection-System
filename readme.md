## Drowsiness Detection System: A prototype

### How to run?

Setup:

```bash
git clone https://github.com/anuragjoshi3519/Drowsiness-Detection-System.git
cd Drowsiness-Detection-System
python3 -m venv env
source env/bin/activate
pip3 install -r requirements.txt

```

Run Web-app:

```bash
python3 app.py
```
